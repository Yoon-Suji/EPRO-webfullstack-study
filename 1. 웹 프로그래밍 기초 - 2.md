### 4) class와 id 속성

* id
  - 고유한 속성, 하나하나에 특별한 제어가 가능하고 검색에 용이

* class
  + 중복해서 사용 가능.
  + 하나의 태그에 여러 개의 다른 class 이름을 공백을 기준으로 나열 가능
  + 전체적인 스타일을 일관성있게 지정하기 위해 사용
